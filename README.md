# tarea3-logs
 La tarea 3 de logs 2024, otoño

## Cómo correr la tarea
* Compilar con la siguiente linea:
```bash
    g++ -std=c++11 main.cpp -o mainPenguin
```
* Correr el ejecutable: 
```bash
./mainPenguin
```

## Cómo correr el graficador
* Ejecutar plotter.py en la terminal

```bash
    python3 plotter.py
```
